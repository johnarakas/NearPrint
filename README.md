## Requirements 

Node.js

## Installation
Server Side

`cd server`

`npm install`

`npm install -g nodemon`

Client Side

`cd client`

`npm install`

`npm install react-bootstrap bootstrap`

## Running
Server side 

`nodemon server`

Client side

`npm start`
